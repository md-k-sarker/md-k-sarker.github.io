The HPC specialization is a 25-credit program combining foundational mathematics, advanced computing concepts, and hands-on learning.

Required Courses (18 Credits)

  • MATH 300 - Differential Equations (3 credits)
  • COSC 381 - Cloud Essentials for Developers (3 credits)
  • COSC 430 - Systems Programming II (3 credits)
  • COSC 466 - System Design and Programming (4 credits) New Course
  • COSC 486 - Parallel and Distributed Computing (3 credits)
  • COSC 492 - CO-OP (3 credits)

Elective Courses (6 Credits)

Choose any two of the following courses:

  • COSC 311 - Data Science
  • COSC 383 - Cloud Applications Development
  • COSC 402 - Software and Operating System Security
  • COSC 439 - Big Data Systems
  • COSC 454 - Embedded System Design
  • COSC 474 - Machine Learning and Discovery
  • COSC 478 - Deep Learning for Visual Recognition
  • COSC 481 - Secure Cloud Applications
